## GitHub  
The first official ReplitConnections community was created on GitHub. Here, members communicate through GitHub Discussions.  
[Visit the GitHub Community](https://github.com/ReplitConnections/ReplitConnections)  

## Discord  
There are several unofficial Discord guilds where ReplitConnections topics are discussed:
1. [YAPC](https://discord.gg/yapc): Yet Another Programming Community (YAPC) is an unofficial Replit Discord server that serves as a hub for programming enthusiasts. With over 16,000 members, it offers a space to discuss coding, participate in programming jams and challenges, and connect with like-minded individuals. The server remains active, providing support and fostering collaboration among its members.

## Discourse  
There are several Discourse communities related to Replit:
1. [AMCForum](https://amcforum.wiki/): AMCForum is an online community that originated from active users of Replit Ask. It serves as a platform for coding enthusiasts to share knowledge, exchange ideas, and collaborate on projects. Beyond coding, AMCForum provides a meeting ground for creative minds to engage in diverse discussions, making it a thriving marketplace of discourse and inspiration.
